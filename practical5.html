Memory Allocation & Fragmentation

Configure memory partitions and processes. Apply First Fit, Best Fit, or Worst Fit allocation strategy. Visualize memory layout and measure internal & external fragmentation.

📚 Theory

Memory is divided into partitions. Allocation strategies decide which partition to assign to a process:

  • First Fit — Allocates the first hole big enough (fastest)
  • Best Fit — Allocates the smallest hole that is big enough (least waste)
  • Worst Fit — Allocates the largest hole (leaves largest remaining fragment)
  • Internal Fragmentation — Wasted space inside an allocated partition
  • External Fragmentation — Free space exists but not contiguous enough to satisfy a request
