📚 Theory

CPU Scheduling decides the order in which processes get CPU time. Key metrics:

  • Arrival Time (AT) — When process enters the ready queue
  • Burst Time (BT) — CPU time required by the process
  • Completion Time (CT) — When process finishes execution
  • Turnaround Time (TAT) = CT − AT
  • Waiting Time (WT) = TAT − BT
