Page Replacement Algorithms

Enter a page reference string, choose number of frames, and simulate FIFO, LRU, or Optimal. Each cell shows if it was a page fault or hit. Compare all algorithms side by side.

📚 Theory

When a process accesses a page not in memory (page fault), the OS must replace a page. Algorithms differ in which page they evict:

  • FIFO — Evicts the page that entered memory first (oldest page)
  • LRU (Least Recently Used) — Evicts the page not used for the longest time
  • Optimal (OPT) — Evicts the page that won't be used for the longest time in future (theoretical best)
  • Hit Ratio = Hits / Total References  ·  Lower page faults = better algorithm
