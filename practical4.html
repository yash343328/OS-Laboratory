Disk Scheduling Algorithms

Simulate FCFS, SSTF, SCAN, and C-SCAN disk scheduling. Visualize the disk head movement on a seek graph. Calculate total and average seek time.

📚 Theory

Disk scheduling determines the order in which disk I/O requests are serviced to minimize seek time (time to move disk head).

  • FCFS — Serve requests in arrival order (simple but inefficient)
  • SSTF — Serve the request closest to current head position (greedy)
  • SCAN (Elevator) — Head moves in one direction, serves all requests, then reverses
  • C-SCAN — Like SCAN but only serves on one sweep, jumps to start on reverse
  • Total Seek Time = Sum of all head movements (in cylinders)
